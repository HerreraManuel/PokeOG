# PokeOG
A discord bot that lets you interact with the original 151 Pokemon.

## Installation
Create a `config.json` file with the bot's token as follows:
```json
{
    "token": "<token>"
}
```
Run the bot and enjoy.


This bot was created for educational purposes.
